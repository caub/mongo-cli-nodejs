Mongo-cli

The API is following nodejs Mongo API, try it in your browser console: 
send({fn:'insert', args:[{a:3}]}, function(d){console.log(d)}) //response broadcasted to all clients connected to this collection
send({fn:'find', args:[{}]}, function(d){console.log(d)}) // return matching documents (for which _canRead list is empty or null, "public documents")
send({fn:'find', args:[{a:{$gt:1.9, $lt:2.1}}]}, function(d){console.log(d)})
send({fn:'findAndModify', args:[{_id:"52bc3f20f170250000983688"},null,{$set:{a:5,b:12}},{new:true}]}, function(d){console.log(d)}) //response broadcasted

_canRead, _canUpsert and _canRemove are special fields only allowed if you're authenticated with an 

send({fn:'insert', args:[{a:4, _canRead:['test.mongocli@gmail.com','j.doe@gmail.com']}]}, function(d){console.log(d)}) //response multicasted to _canRead group
send({fn:'find', args:[{a:4}]}, function(d){console.log(d)}) //returns matching documents for which _canRead list is empty, null or containing your identity
send({fn:'remove', args:[{a:4}]}, function(d){console.log(d)}) //removes matching documents for which _canRemove list is null, empty or containing your identity
