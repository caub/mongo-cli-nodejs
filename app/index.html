Mongo-cli

Commands (try them in your dev-console) are following nodejs Mongo API 
send({fn:'insert', args:[{a:3}]}, function(d){console.log(d)}) //is broadcasted
send({fn:'find', args:[{}]}, function(d){console.log(d)})
send({fn:'find', args:[{a:{$gt:1.9, $lt:2.1}}]}, function(d){console.log(d)})
send({fn:'findAndModify', args:[{_id:"52bc3f20f170250000983688"},null,{$set:{a:5,b:12}},{new:true}]}, function(d){console.log(d)}) //is broadcasted

_canRead, _canUpsert and _canRemove are special fields only allowed if you're authenticated with an 

send({fn:'insert', args:[{a:4,_canRead:['cyril.auburtin@gmail.com']}]}, function(d){console.log(d)})
send({fn:'find', args:[{a:4}]}, function(d){console.log(d)}) //returns it if you're this user ^, try with your email
