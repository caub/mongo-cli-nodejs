MongoWebConsole-nodejs
======================

MongoDB from a websockets, the syntax is following nodejs-mongodb driver API: http://mongodb.github.io/node-mongodb-native/api-generated/collection.html

example in javascript: `send({fn:'find', args:[{name: 'cyril'}]}, function(d){console.log(d)})`

demo: https://c9.io/cytr/demo-project/workspace/html/index.html
